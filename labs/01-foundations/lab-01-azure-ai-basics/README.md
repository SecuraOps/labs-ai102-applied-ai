# Lab 01 – Azure AI Basics

**Difficulty:** Beginner  
**Estimated time:** 60–90 minutes  
**AI-102 Domains:**  
- Plan and manage Azure AI solutions  
- Implement Azure Cognitive Services  

## Goal
Understand the fundamentals of Azure AI services by:
- Provisioning an Azure AI resource
- Securing access using keys
- Calling the service from real code

This lab establishes the baseline used throughout the rest of the repository.
